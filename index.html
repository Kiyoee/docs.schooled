Information

Welcome to schooled application this application is for a better and easy school experiance for students with the help of schooled you can do a lot of things for a easier school life that is listed below

Homework Tracking

Using the schooled application you can do homework tracking easily when there is a new homework you can add that to your homeworks tab with only one click and you can enter more details like category, time, level and much more when the homework time is done you would take a in-app notification for that

Customization

We let every student customize the application for a better experiance for both sides everytime you do a action there would be customization options for a colourful experiance

Chat

On our application there is a "Chat" tab that teachers and students can use to talk with eachother the usage is really simple just click on it and start chatting with this tab students can talk with eachother about the class, homeworks and more and teachers can announce new homeworks online and chat with students about the homeworks

Discord Integration

Our application is partnered with Discord Education Servers with the app you can get a special discord server for your class that you can announce new homeworks, projects, updates and more

Projects

On our application there is a "Projects" tab that students and teachers can add new projects to the class and students can control their projects using our application very easy

Timing

Our application is coming with a new "Timing" feature that students can make a program about their homeworks and projects teachers can give timings to students and they can edit the homework to special times that could help them with homeworks and they being more effective on projects
